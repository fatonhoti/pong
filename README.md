## Getting Started

The following instructions should get a demo going on your PC.

### Installation

1. Clone the repo

   ```
   git clone https://github.com/fatonhoti/pong.git
   cd pong
   ```
3. Install relevant packages
   ```
   pip install -r requirements.txt
   ```
4. Run the game
   ```
   python pong.py
   ```
